# The Luxury Pro Hotel Management App

## Front-end React App used only in side the Hotel by the employees to manage customers.

### Features

- Authentication integrated via Supabase API
- Employees are able to Check-in/Check-out customers
- Employees are able to create new Bookings and even create new Cabins
- Employees are able to Update their profile
- With Darkmode implemented to it

### Technologies used

- React
- React-query for fetching remote data
- React-router for navigation
- Supabase (as database and API)
- Styled components for styling
- And a lot more other libraries

### Some improvements you may want to add

- Make it responsive as it is designed only for desktop

### Want to run this app locally

- Clone the repositoy `git clone https://github.com/fike-dev/the-luxury-pro.git`
- Install dependecies `npm i`
- Run locally `npm run dev`
